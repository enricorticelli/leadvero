## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→run lifecycle API | Untrusted run identifiers and stop/start requests enter orchestration layer |
| queue worker→database | Asynchronous worker writes influence user-visible totals and status |
| server→external public sources | Outbound calls must follow policy/rate limits to avoid abuse/compliance issues |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-05 | D | `src/server/discovery/orchestrator.ts` | mitigate | Apply per-run caps and bounded queue concurrency to prevent oversized discovery jobs. |
| T-01-06 | R | `src/server/discovery/providers/index.ts` | mitigate | Persist provider call metadata and terminal run status for auditable discovery traceability. |

</threat_model>
