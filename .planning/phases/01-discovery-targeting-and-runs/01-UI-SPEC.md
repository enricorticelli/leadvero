## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: icon-only controls may use 44px minimum touch target (default)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F8FAFC | App background and primary surfaces |
| Secondary (30%) | #E2E8F0 | Cards, form panels, run monitor sections |
| Accent (10%) | #0F766E | Primary CTA, active progress bar, selected platform filter chip, focused primary input ring |
| Destructive | #B91C1C | Stop-run confirmation actions only |

Accent reserved for: `Start Discovery Run` button, active/in-progress status indicators, selected platform target control, and current-run progress visualization only.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Start Discovery Run |
| Empty state heading | No candidates discovered yet |
| Empty state body | Launch a run with keyword, country, and platform filters to start collecting public business domains. |
| Error state | Discovery run could not continue. Check filters or source availability, then retry the run. |
| Destructive confirmation | Stop discovery run: This will stop new candidates from being added. Existing discovered domains will be kept. Continue? |
